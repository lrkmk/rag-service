# FR 集成

使用此页面进行瑞安航空（FR）集成。

### 概述

FR 集成涵盖瑞安航空特定的预订要求，包括在支付前需要订单确认（`orderCommit.do`）。

### 关键区别

* FR 预订在支付前需要 `orderCommit.do`
* FR 有特定的旅客信息要求
* FR 的附加服务规则与其他航司不同
