# PNR 认领与提取

使用此页面认领和提取 PNR 数据。

### 概述

PNR 认领允许您将现有的航司 PNR 关联到 Atlas 系统。PNR 提取允许您从 Atlas 订单中检索 PNR 数据。

### 常见用例

* 将外部创建的 PNR 关联到 Atlas
* 从现有 Atlas 订单中提取 PNR 详细信息
* 同步航司和 Atlas 之间的 PNR 状态
