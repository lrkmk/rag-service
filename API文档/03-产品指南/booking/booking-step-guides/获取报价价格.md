当您需要 `getOfferPrice.do` 履约流程时，使用此页面。

此流程专为已知道目标订单上下文且需要 Atlas 在严格的出票窗口内完成快速履约的场景而设计。

### 常见问题

#### 何时应使用 `getOfferPrice.do`？

当您需要具有更广泛展示规则和快速出票截止时间的履约流程时，使用 `getOfferPrice.do`。

此流程适用于紧急出票、高风险的库存情况以及被标准获取报价展示策略排除的场景。

#### 此流程与 `getOffers.do` 有何不同？

`getOfferPrice.do` 是面向履约的路径。

它允许更广泛的报价可见性，使用自己的请求限制策略，并在订单创建后应用 5 分钟履约窗口。

#### 此流程放宽了哪些限制？

此流程可以返回通常在标准获取报价路径中被过滤的报价。

放宽的规则包括：

* 临近出发的航班
* 售罄风险场景
* 高于 `OW + OW` 的往返价格

#### 支付和出票截止时间是什么？

从此流程创建的订单具有 5 分钟支付和出票窗口。

如果出票未及时完成，Atlas 将自动取消订单。

### 主要 API

* `getOfferPrice.do`
* `getLuggage.do`
* `seatAvailability.do`
* `order.do`
* `pay.do`
* `queryOrderDetails.do`

### 此流程的不同之处

#### 更广泛的报价可见性

此流程可以显示标准 `getOffers.do` 可能隐藏的库存。

这使得它对紧急或难以履约的场景很有用。

#### 独立的请求治理

`getOfferPrice.do` 使用自己的 QPM 策略。

不要假设它与 `verify.do` 或 `getOffers.do` 共享相同的池。

#### 更快的操作截止时间

履约窗口比标准预订保留时间短得多。

规划完整链以实现即时支付和主动跟进。

### 5 分钟履约规则

Atlas 期望在此流程中快速完成整个履约链。

使用以下规则：

* 仅在准备好支付时才创建订单
* 立即完成支付，不要延迟
* 持续轮询直到最终出票或最终失败
* 当订单已超出允许窗口时停止重试

{% hint style="warning" %}
不要将此流程视为标准订单保留流程。

操作窗口是 5 分钟，而不是 30 分钟。
{% endhint %}

### 重试和超时处理

在每次重试前检查剩余时间。

作为工作规则，仅在订单仍在安全的 4 分钟操作窗口内时重试。

如果订单已接近超时，避免另一次支付重试，先查询最新的订单状态。

如果出票在 5 分钟内仍未完成，Atlas 将自动取消订单。

### 订单标识和恢复

从此流程创建的订单应保留其履约标识。

如果您需要在支持的取消路径后重新生成订单，在重新生成的订单上保留相同的履约流程标记。

仅当原始订单不再可支付且恢复路径支持重新生成时，使用 `regenerateOrder.do`。

### 航司范围

默认范围是所有航司。

U2 和 9C 目前被排除在此流程之外。

它们无法可靠地满足 5 分钟出票要求。

对于 U2 和 9C，请改用标准预订路径。

### 典型流程

{% stepper %}
{% step %}

### 获取履约报价

调用 `getOfferPrice.do` 并保留返回的 `OfferId`。
{% endstep %}

{% step %}

### 在需要时添加附加服务

当座位或行李追加销售是您产品的一部分时，在订单创建前查询 `getLuggage.do` 或 `seatAvailability.do`。

在此流程中使用返回的 `OfferId` 作为附加服务上下文。
{% endstep %}

{% step %}

### 创建订单

仅在支付可以立即开始时，使用 `OfferId` 调用 `order.do`。

将订单创建视为严格履约窗口的开始。
{% endstep %}

{% step %}

### 支付订单

一旦订单准备就绪，立即调用 `pay.do`。

在此流程中不要延迟支付。
{% endstep %}

{% step %}

### 跟进出票到最终状态

使用 `queryOrderDetails.do` 直到订单出票或取消。

Webhook 可以提供帮助，但不应该是您唯一的确认路径。
{% endstep %}
{% endstepper %}

### 最佳实践

* 仅将此流程连接到快速支付用户体验
* 保持重试逻辑保守
* 使用 `orderNo` 记录履约流程标记
* 对超时取消和支付失败进行分开监控
* 使用符合 5 分钟 SLA 的航司允许列表或阻止列表
