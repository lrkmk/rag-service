# 行李

使用此页面查询和管理行李附加服务。

### 概述

行李 API（`getLuggage.do`）允许您在预订前查询行李选项和价格。

### 关键规则

* 在确认目标报价后查询行李
* 使用 `sessionId`（标准流程）或 `OfferId`（获取报价流程）
* 在订单创建前查询行李以提高旅客清晰度
