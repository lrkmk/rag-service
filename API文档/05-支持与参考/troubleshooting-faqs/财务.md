使用本页面了解 Atlas 财务操作，包括余额、充值、结算和退款对账。

当你需要以下内容时，从这里开始：

* 检查余额或充值账户
* 对账预订后交易
* 了解退款资金去向

### 常见问题

#### 如何查询余额？

使用 `queryBalance.do` 或访问 ATRIP 中的 `Finance` -> `Balance`。

### 如何查询余额？

使用 `queryBalance.do` 或访问 ATRIP 中的 `Finance` -> `Balance`。

#### 如何充值？

ATRIP 中的 `Finance` -> `Top Up`。

选择存款金额并按照流程操作。

### 如何充值？

使用 ATRIP 中的 `Finance` -> `Top Up`。

选择存款金额并按照流程操作。

#### 如何对账？

使用 ATRIP 中的 `Finance` -> `Settlement`。

### 如何对账？

使用 ATRIP 中的 `Finance` -> `Settlement`。

#### 结算频率是多久？

每日结算。

### 结算频率是多久？

每日结算。

#### 资金如何流动？

对于 **VCC 直通**，航空公司直接从您提供的卡中收取机票费用。Atlas 仅从您的 Atlas 余额中扣除技术服务费。

对于 **存款**，Atlas 从您的余额中扣除总订单金额，并使用 Atlas 存款向航空公司支付。

#### 退款资金记入哪里？

对于 VCC 直通，退款资金通常返回到原始卡。

对于存款，Atlas 在收到航空公司资金后记入您的余额。

#### 如何处理退款对账？

退款在 `Finance` -> `Settlement` 中可见。

使用 ATRIP 中的结算工具追踪退款状态。

#### 何时能收到退款？

退款时间因航空公司而异。

一旦 Atlas 收到航空公司退款，余额更新反映在您的账户中。
