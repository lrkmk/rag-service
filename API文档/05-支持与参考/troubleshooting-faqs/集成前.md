### 常见问题

#### 集成前的检查清单是什么？

在开始实施前，您应该：

1. 确认商业协议已签署
2. 确认已获取沙箱凭证
3. 确定首选支付模式（存款、VCC 直通或两者）
4. 确定目标航空公司
5. 确认流程选择（标准搜索、Get Offer 或两者）

#### Atlas 适合所有类型的旅行社吗？

Atlas 支持 OTA、TMC 和旅行社。

在开始实施前，确认您的商业模式与 Atlas 产品一致。

#### 是否需要专门的开发人员？

是的。您需要后端开发人员将 API 调用集成到您的系统中。

#### 从开始到上线需要多长时间？

集成时间取决于范围。简单的预订流程可以在几周内完成，而支持退款和附加服务的完整集成可能需要数月。

是的。\

#### Atlas 是否需要 NDA？

是的。\
在提供沙箱凭证之前，Atlas 要求签署 NDA。

#### Atlas 是否支持所有航空公司？

Atlas 连接 150 多家航空公司。\
您可以在[航空公司列表](https://www.atriptech.com/#/airline/list)和 ATRIP 中查看当前列表。

#### 我们可以销售哪些类型的票价？

Atlas 支持标准票价和促销票价。\
促销代码支持正在规划中。

#### Atlas 是聚合器还是 GDS？

Atlas 是一个聚合平台，通过 API 直连 150 多家航空公司。\
它不是 GDS。
