当航空公司要求在请求中使用受支持的区域设置值时，使用本页面。

### 本页面涵盖的内容

本页面按航空公司列出了支持的区域设置值。

在航空公司流程要求时，在 `locale` 等请求字段中使用这些值。

### 区域设置何时重要

区域设置在以下情况最为相关：

* 航空公司需要特定市场的语言或国家代码
* 预订流程显示航空公司托管的确认或结账内容
* FR 特定的订单流程需要区域设置值

### 使用位置

区域设置用于：

* [创建订单](/api-wen-dang/product-guides/booking/booking-step-guides/create-order.md)
* [FR 集成](/api-wen-dang/product-guides/extensions-and-integrations/special-integrations/fr-integration.md)

### 提示

* 仅使用为该航空公司发布的值
* 不要假定区域值可以与请求区域设置值互换
* 保持区域设置与您向用户展示的特定航空公司流程一致
