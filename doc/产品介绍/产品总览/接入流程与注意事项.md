# 接入流程与注意事项

Atlas 提供从注册到上线的标准接入路径。

建议先在沙箱完成联调，再进入生产验证。

### 快速接入四步

{% stepper %}
{% step %}

### 注册

提交企业名称、联系人信息和邮箱验证。

通常 1 分钟内完成注册。
{% endstep %}

{% step %}

### 免费测试

注册后自动生成沙箱密钥。

建议先发起一次搜索，再走通首次 API 调用。
{% endstep %}

{% step %}

### 上线

完成邮箱验证、选择套餐并支付首月费用。

生产密钥签发后，即可开始正式联调。
{% endstep %}

{% step %}

### 出票

搜索能力随套餐提供。

开始销售前，请准备预充值余额并完成支付路径验证。
{% endstep %}
{% endstepper %}

### 环境说明

#### 沙箱环境

适合开发联调、流程验证和异常场景模拟。

地址：`https://sandbox.atriptech.com/`

#### 生产环境

生产环境包含独立的搜索服务和业务接口。

具体地址可在 ATRIP 后台查看。

### UAT 验证要求

航班预订流程是必测项。

至少需要完成：

* 搜索
* 验价
* 创建订单
* 支付出票

退款、废票、改期、Webhook 和邮件服务可按实际集成范围验证。

{% hint style="warning" %}
提交 UAT 时，必须包含航班预订能力验证。
{% endhint %}

### 上线前重点确认

* 响应字段与状态以真实返回为准
* Webhook 结构需真实接收验证
* 支付成功、失败和重试路径需完整测试
* 沙箱与生产环境可能存在差异

### 服务保障与边界

#### 服务保障

* ≥90% 订单在 5 分钟内完成出票
* ≥98% 订单在 1 小时内完成出票
* 所有退票申请 4 小时内完成受理并提交航司

#### 服务边界

* Webhook 以尽最大努力方式送达
* 航变请以航司发送到预订邮箱的邮件为准
* 废票服务可能因航司政策、风险控制或业务调整而变化

### 推荐的上线前检查清单

* 已完成搜索到出票的完整联调
* 已测试支付成功、失败和重试路径
* 已验证 Webhook 接收和重试处理
* 已确认沙箱与生产环境参数差异
* 已准备售后处理流程和联系人机制
