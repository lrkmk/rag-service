# 爱琴海航空 Aegean Airlines (A3) 与奥林匹克航空 Olympic Air (OA) 服务升级，强化出票后服务能力

在舆途科技（Atlas），我们的目标始终清晰明确：实现全面航司覆盖、真正的端到端服务、高数据准确率，以及为旅游销售商提供更简化、更自动化的工作流。&#x20;

本次爱琴海航空 Aegean Airlines（A3）与奥林匹克航空 Olympic Air (OA) API 能力升级，是这一长期承诺的又一次重要落地。在进一步增强出票后能力的同时，也有效降低了整体运营复杂度。&#x20;

#### **舆途科技（Atlas）API 中A3和OA的新功能：**&#x20;

📅 可售周期：最长支持 90 天提前预订&#x20;

🎟️ 全部票价产品类型已完整支持&#x20;

🧳 支持随单行李&#x20;

💳 支付方式：支持 VCC 支付&#x20;

🔁 出票后自动化能力（新增）：自动化退票和自动化废票&#x20;

通过将自动化能力直接嵌入出票后操作流程，本次升级将帮助旅游销售商提升运营效率，同时为旅客带来更加顺畅、稳定的出行体验。&#x20;
