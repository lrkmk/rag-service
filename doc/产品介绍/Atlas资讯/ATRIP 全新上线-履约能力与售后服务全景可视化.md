# ATRIP 全新上线：履约能力与售后服务全景可视化

ATRIP 现已上线两大全新页面，全面提升履约与售后信息的透明度——从出票到退票，全流程一目了然。

无需反复确认，无需频繁跟进。所有与订单履约及售后处理相关的信息，现在都可以在一个平台内清晰掌握。

### **一、履约能力页面**

**下单前，即可清晰预期履约表现**

全新上线的履约能力页面，集中展示 Atlas 在平台上 **140+ 航司的出票与退票履约能力**，帮助您在下单前做出更有把握的决策。

**您可以快速查看：**

* 平台整体出票 SLA ：≥90% 订单在 5 分钟内完成出票；≥98% 订单在 1 小时内完成出票
* 退票受理承诺 ：所有退票申请将在 4 小时内完成受理并提交航司（其中 95% 的退票申请可在 15 分钟内完成受理）
* 各航司出票渠道：API / B2B Portal / 航司官网等
* 各航司退票渠道：API / B2B Portal / 官网订单管理 / 邮件等
* 不同支付方式的出票保障差异：余额支付 vs VCC支付

您还可以点击任意航司，查看其详细能力，包括：

* 出票渠道
* 退票渠道
* 退票处理周期
* 航司退票规则

### **二、售后服务报表**

**实时追踪每一笔退款与工单，无需反复询问**

全新售后服务看板为您提供订单售后处理的实时视图，让每一个退款与服务请求的进度都清晰可见。

{% hint style="warning" %}
**该功能仅对已上线的正式客户开放**
{% endhint %}

**您可以实时追踪：**

* **退票请求：** 覆盖各阶段状态，Atlas 处理中 / 航司处理中 / 航司退款中 / 已完成 / 已拒绝 / 已取消
* **服务请求**：通过 EVA 提交的所有工单的实时状态
* **航司退票处理与退款时效基准**：按航司维度统计中位数 / 平均值 / 最大时长，支持区分起飞前退票 / 起飞后退票

通过这些数据，您可以更准确地预估退款到账时间，减少不必要的沟通与跟进。

### **三、使用场景**

无论是**下单前决策**，还是**订单执行与售后管理**，这两大页面都能帮助您让业务更加可控、可预期。

**下单前-使用**「 **履约能力**」**页面**

您可以：

* 提前了解不同航司的出票速度与履约能力，合理设定客户预期
* 查看各航司支持的退款渠道（系统将自动选择最优路径）
* 理解不同支付方式（预存款 vs VCC）的出票保障差异，优化支付策略

**下单后-使用**「 **售后服务报表**」**页面**

您可以：

* 实时查看每一笔退款进度，无需再通过邮件或IM反复跟进
* 跟踪通过 EVA 提交的服务请求处理状态
* 参考航司退款时效基准，管理客户对退款周期的预期，并及时识别异常延迟

这两大页面将帮助您实现：

✅ 更少的履约不确定性

✅ 更少的跟进与沟通成本

✅ 更高的客户沟通信心与专业度
