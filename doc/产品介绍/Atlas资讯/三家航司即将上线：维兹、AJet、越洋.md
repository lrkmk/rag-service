# 三家航司即将上线：维兹、AJet、越洋

舆途科技（Atlas）航司数据版图迎来新一轮扩展。**维兹（Wizz Air， W6 / W4 / W9 ）、AJet（VF）**&#x548C; **越洋（Air Transat，TS）**&#x7684; API 数据即将在舆途科技（Atlas）正式上线。

通过一个 API，即可进一步扩大您在欧洲、土耳其及加拿大市场的航司覆盖能力，获取更多廉航库存、辅营销售机会及跨境连接方案。

### 一、维兹（Wizz Air， W6 / W4 / W9 ）

作为欧洲最大的超低成本航司（ULCC）之一，**维兹**年运送旅客超过 7000 万人次，运营超过 750 条航线，覆盖 50+ 国家和地区。

其网络重点覆盖：

* 中东欧
* 巴尔干地区
* 中亚
* 中东
* 北非

对于希望拓展欧洲二三线城市、高性价比自由行产品的合作伙伴而言，维兹是极具竞争力的库存补充。

**通过舆途科技（Atlas），合作伙伴可获得：**

* 全面直飞航线覆盖，预订全流程数据稳定
* 支持辅营产品嵌入式销售：选座、托运行李
* 上线初期提供 90 天预订窗口，180 天窗口扩展正在评估中
* 定价与航司官方渠道保持一致

### 二、AJet（VF）

舆途科技（Atlas）是首家完成 **AJet NDC v24.1** 对接并正式上线的合作伙伴。

**AJet** 隶属于土耳其航空集团（Turkish Airlines，TK），依托土耳其航空的全球枢纽网络，正在快速扩展其区域及国际航线能力。

对于中国市场而言，土耳其长期是连接欧洲、中东及部分中亚市场的重要中转节点。AJet 的加入，将进一步丰富合作伙伴在土耳其及周边区域的廉航库存覆盖。

**通过舆途科技（Atlas），合作伙伴可获得：**

* 土耳其境内及国际航线完整库存数据
* 自上线首日起支持辅营销售：行李、选座、票价组合产品
* 定价与航司官方渠道保持一致

### 三、越洋（Air Transat，TS）

**越洋**是加拿大领先的休闲航空公司，以加拿大至欧洲的跨大西洋网络著称，同时覆盖加勒比、墨西哥及拉丁美洲等热门度假市场。

虽然越洋不属于LCC，但其强大的辅营能力（Fare Families、选座、行李等）正是舆途科技（Atlas）智能基础设施擅长处理的核心场景。我们可为这类航司提供与LCC同等的标准化集成与动态定价支持，帮助合作伙伴在长线休闲航线上获得更高的辅营渗透率和订单价值。

**这意味着：**&#x66F4;多可售库存、更多升销机会、更完整的端到端廉航分销流程，最终带来更高的转化率和客单价。
