# 保护航班：信息查询与后续操作指南

若您预订的国际航班因航司原因（如航班取消、时刻变更）产生保护航班，本指南将为您介绍具体的确认方式及后续操作，助您高效处理客票售后问题。

## 一、核心原则：廉航保护航班无需额外确认&#x20;

廉航（LCC）的保护航班，通常无需手动确认或操作，乘客可直接凭原购票证件（如护照）在保护航班起飞前，前往机场航司柜台办理值机并乘坐。航司会主动安排保护航班，无需担心因“未确认”而影响行程。&#x20;

## 二、保护航班信息的获取方式&#x20;

航司一般会通过邮件将保护航班的具体信息（如航班号、起降时间、舱位等）发送至您出票时预留的邮箱。您可通过以下方式查看：&#x20;

* 若您使用了 Atlas 邮箱服务：请登录 Atrip 系统，进入对应订单详情页面，在【邮件列表】 模块中查找航司发送的保护航班通知邮件。&#x20;
* 若您未使用 Atlas 邮箱服务：请直接登录您出票时预留的邮箱，查找来自航司官方域名（如xxx.airlines.com）的邮件，邮件主题通常包含“schedule change”、“flight update”等关键词。&#x20;

{% hint style="warning" %}
**注意：所有保护航班信息以航司官方邮件为准，无需通过其他渠道二次核实。**&#x20;
{% endhint %}

## 三、如对保护航班不满意，可参考航司方案处理&#x20;

若乘客对航司安排的保护航班不满意（例如时刻不合适），可以根据航司邮件中提供的其他方案进行后续操作，通常包括：&#x20;

* **退票**：部分航司会在邮件中提供 “自助退票链接”，或说明退票申请方式（如联系航司客服、通过原购票渠道提交）。&#x20;
* **改签**：航司可能会列出可改签的其他航班选项，您可按邮件指引选择新航班并完成改签（部分操作可能需通过航司客服协助）。&#x20;

{% hint style="warning" %}
**注意：具体可选方案及操作流程，请务必以您收到的航司官方邮件为准。**&#x20;
{% endhint %}
