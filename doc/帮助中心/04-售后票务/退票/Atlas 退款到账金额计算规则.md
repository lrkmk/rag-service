# Atlas 退款到账金额计算规则

Atlas 将根据从航司实际收到的退款金额，并按公允汇率折算后，为您办理退款。

具体规则如下：

### **1.全额退款**

**适用情况**：在非自愿退票、废票等场景中，航司已全额退还现金。

**计算规则：** Atlas 将以订单成交价为基础，按您原支付的币种和实际支付金额进行全额退款。 其中，订单成交价指您在下单时，实际向Atlas支付的订单金额。&#x20;

**举例说明：** 若您下单时订单成交价为 1000 CNY，且该订单符合全额退款条件、航司已完成全额退款，Atlas 将向您退还 1000 CNY。&#x20;

### **2.部分退款**

**适用情况**：在自愿退票、退税、非自愿退票等场景中，航司收取退票手续费或仅同意退还部分金额。

**计算规则**：Atlas 以实际从航司收到的退款金额为基准，根据退款操作前一日的汇率，将航司退款币种折算为您原付款的币种后，退还至您的账户。

**举例说明**：您下单支付了1000CNY，申请退款后，航司根据规则扣除手续费，实际退给我司50USD。假设退款操作前一日的汇率为1USD = 7.0 CNY，则折算金额为50USD ×7.0= 350 CNY。因此，Atlas 将向您退还350CNY。

{% hint style="warning" %}
退款金额中不包含原订单的出票服务费。该费用为出票时已产生的成本，因此无法退还，感谢您的理解。
{% endhint %}
