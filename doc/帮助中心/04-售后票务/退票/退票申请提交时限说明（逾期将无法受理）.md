# 退票申请提交时限说明（逾期将无法受理）

退票申请的最晚提交时限，并非统一标准，而是根据具体退票类型及航司规则确定。您在提交申请时，Atlas 系统将自动进行判断与校验。&#x20;

最晚提交时限以航司 Fare Rule 规定与客票有效期（通常为一年，未开始旅行按出票日计算，已开始旅行按首段航班起飞日计算）中较严格的为准。&#x20;

如您有退票需求，建议尽早提交申请。系统将在提交时自动进行时限校验，超出时限的申请将无法受理。&#x20;
