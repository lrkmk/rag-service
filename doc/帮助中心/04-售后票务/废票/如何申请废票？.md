# 如何申请废票？

您可以通过ATRIP申请废票。

**废票路径：**&#x41;TRIP -> 订单中心 -> 我的订单 -> 订单详情 -> 废票

{% hint style="warning" %}
若页面中没有废票入口，则说明Atlas暂不支持该航司的废票服务。
{% endhint %}

![](/files/zsYJ5fFtZWXhSMP9iPOk)

**操作说明：**

1\. 点击"废票" 按钮。

2\. 确认需要废票的乘客及航段信息后，选择废票/退款方式。Atlas支持以下两种方式：

* 如果您需要Atlas代您向航司提交废票申请，请选&#x62E9;**【尚未操作，请Atlas操作废票】 ，**&#x41;tlas将为您提供废票服务。<mark style="color:red;">**注意：废票申请提交后将无法撤回，请谨慎操作。**</mark>
* 如果您或乘客已自行向航司提交了废票申请，并且得到了航司的确认，请选&#x62E9;**【航司已废票，请Atlas退款】 ，**&#x41;tlas将在收到航司的退款后立即为您办理退款。

{% hint style="warning" %}
对于使用VCC Passthrough支付方式的订单(可以通过订单详情里的“支付渠道：虚拟信用卡”来辨识），不支持选择“**航空公司已退票，请Atlas退款**”选项。因为在客户使用自己的信用卡支付时，信息传递只发生在客户和航空公司之间，Atlas无法了解实际退款金额的到账情况。
{% endhint %}

![](/files/qwHGTYMVCirZWxmeqLBu)

**注：**

* 仅支持按PNR维度提交废票，不支持为订单里的部分乘客提交废票申请；
* 若当前不在废票窗口期内，则无法提交废票申请；
* 若订单中有部分乘客已经提交了其他原因的退票申请，则无法提交废票申请。

3\. 提交废票/退款申请

1）若选择【尚未操作，请Atlas操作废票】，页面中将会展示对应的废票信息，点击“确认”按钮，即可提交废票申请。

![](/files/vc96qoaAEsZGtN2vg0Qp)

2）若选择【航司已废票，请Atlas退款】，请根据页面指引填写履约日期（即航司确认废票的日期）、上传航司履约证明，然后点击“确认”按钮，提交退款申请。

![](/files/RMcCLLEg6TizVzfrSIrW)

4\. 提交成功后，Atlas将收到并处理您的废票/退款申请，同时您的申请记录会展示在“退票信息”和“退票单”页面，您随后可以在任意一个页面跟踪处理进度。
