# Atlas改乘客信息政策说明

## 一、常规处理原则

Atlas 通常不直接受理改乘客信息（如改乘客姓名、护照号等）服务，主要原因说明如下：

1. 身份验证要求：航司在处理信息修改时，通常需乘客提供护照等身份证明文件进行核验，由官方渠道直接处理更为稳妥。
2. 操作流程复杂：乘客姓名等信息格式要求严格，多环节经手易因拼写、格式不符等细节出现错误。
3. 处理时效保障：Atlas 作为非官方渠道，介入需额外增加中转环节，会拉长处理周期；而航司直办可直接对接核心系统，流程更短、响应更快，能避免因时效问题影响出行。

{% hint style="warning" %}
建议您直接通过对应航空公司的官方网站或客服呼叫中心查询相关政策并办理相关手续，以确保信息同步及时、操作高效准确。
{% endhint %}

## 二、特殊情况协助服务

仅在特殊场景下，如航司仅允许通过机票代理商（Atlas）办理改乘客信息服务（适用航司清单见下文）。Atlas 可协助处理更改事宜，但需满足以下条件：

1. 申请时间：需在航班起飞前至少24小时提出。
2. 处理结果：Atlas 不保证一定更改成功，仅在航司确认的前提下方可完成。
3. 服务费用：改乘客信息需收取服务费，具体标准请参照您与 Atlas 签署的合同。

## 三、Atlas 可协助的航司清单

### 1. 改姓名场景

“改姓名”在此特指姓名更正，即修正机票上的姓名填写错误，使其与乘客身份证件保持一致，而非更换乘机人。具体区别如下：

* **姓名更正**：
  * 指修正拼写错误、漏字/多字、姓与名顺序颠倒、中间名增删等情况，以确保机票姓名与乘机人身份证件一致。&#x20;
  * 示例：JHON → JOHN&#x20;
* **姓名变更**：
  * 指更换乘机人，即原乘客被另一人替代，不属于本场景支持的姓名修改范围。&#x20;
  * 示例：John Smith → David Brown&#x20;
