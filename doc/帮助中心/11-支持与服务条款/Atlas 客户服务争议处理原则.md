# Atlas 客户服务争议处理原则

## 1. **服务违规定义**

Atlas将为因自身运营问题导致无法出票的情况提供补救措施，但不包括因规则解释模糊引发的争议。

| 违规类型 | 责任  | 定义                               |
| ---- | --- | -------------------------------- |
| 履约失败 | 可赔付 | Atlas 系统/操作错误导致订单失败（如航班售罄、出票错误等） |
| 服务争议 | 不赔付 | 因规则模糊或解释差异引发的争议（如退票费用、履约时限等）     |

## **2. 可赔付场景及赔付标准**

### **2.1 航班售罄**

**触发条件**：付款后因航班售罄无法履约。\
**赔付标准**：\
(a) 提供同服务等级的其他航班；**或**\
(b) 全额退款 + **每位乘客补偿50 USD**。

### **2.2 系统 / 操作故障**

**触发条件**：Atlas错误导致无法出票（不含航班售罄情况）。\
**赔付标准**：\
(a) 通过外部渠道重新出票（同航班及服务等级） ；**或**\
(b) 全额退款 + **每位乘客补偿50 USD** 。

### **2.3 到场无票（Atlas责任）**

**触发条件**：因Atlas过错导致乘客无法凭有效证件值机，且Atlas未提前检测或通知异常情况。\
**赔付标准**：\
(a) 现场重新购买同服务等级的新票 + **每位乘客补偿50 USD** ；**或**\
(b) 全额退款 + **每位乘客补偿50 USD** 。

### **2.4 服务信息不符**

\
**触发条件**：实际出票与订单要求不符（如行李额度、座位分配等）\
**赔付标准**：补偿经核实的损失差额 + **每位乘客补偿50 USD**&#x20;

{% hint style="warning" %}
注：赔付提交时限为航班起飞后10个工作日内。若您符合上述赔付条件，可提交“争议”工单，Atlas 将安排专人处理。
{% endhint %}

## **3. 不可赔付场景及说明**

### **3.1 出票时效争议**

**(a) Atlas常规票务处理效率：**

* ≥90% 订单 在5 分钟内出票
* &#x20;≥98% 订单 在1 小时内出票

\
\&#xNAN;**(b) 若对“出票中”状态的特定订单时效不满**，可选择：

* 取消预订（需承担后续责任）；**或**
* 继续等待（系统将完成出票）。

### **3.2 航班变更（改期）争议**

(a) 仅支持指定航空公司；非指定航空公司需直接联系航司。\
(b) 处理时效：

* 自愿改期：4 小时内完成（特殊情况立即通知）
* &#x20;非自愿改期：遵循航司政策

(c) 不适用于起飞前 48 小时内。\
(d) 撤销改期请求：Atlas将协助，但成功与否由航司决定。

### **3.3 退票费争议**

(a) Atlas的退票费报价仅供参考。\
(b) 实际退款金额以航司最终核退为准。\
(c) Atlas承诺：

* 100% 转交航空公司退款项（包括拒退、未明示费用或汇率差异）
* 按合同收取固定处理费。

### **3.4 航班中断**

(a) 航司关于航班取消或时刻变更的通知将发送至下单时提供的邮箱。确保邮箱准确性是您的责任。若使用Atlas邮箱服务，请参考相关条款。\
(b) 若乘客根据欧盟第 261/2004 号条例向航司索赔，而航司因此向Atlas索赔，您需协助Atlas提供收到通知的证据。Atlas可能会从您的保证金账户中扣除航司索赔的全额款项。

### 3.5 拒绝登机与超售&#x20;

一旦完成预订，即视为Atlas同意遵守实际承运航空公司的《运输总条件》、《运价规则》及超售相关条款。作为机票销售方（卖方），须遵循相同条款，仅可对必要内容进行调整。实际承运航司对旅客是否符合补偿条件、改签资格及相关协助（如餐食、住宿）享有最终决定权，适用法律包括但不限于欧盟第 261/2004 号条例及美国交通部规章。卖方负有独家责任向旅客传达实际承运航司的具体规则、值机截止时间及拒载程序，并确保旅客知悉并遵守相关要求。&#x20;

&#x20;

### 3.6 航变未通知&#x20;

(a)客户未使用Atlas邮箱服务。&#x20;

(b)客户虽已使用Atlas邮箱服务，但因航司、机场等外部原因（非 Atlas 责任）导致未接收到航变邮件通知。&#x20;

### 3.7 客户自有虚拟信用卡支付异常&#x20;

(a)支付失败&#x20;

(b)支付遗漏&#x20;

(c)支付风控校验导致客票异常&#x20;

&#x20;

### 3.8 出票差价&#x20;

为确保价格透明，除儿童客票因适用特定折扣所产生的差价外，Atlas 原则上不承担因票价实时波动、航司价格调整、渠道价格差异、汇率变化、库存变化或其他非 Atlas 原因导致的出票后差价退还责任。&#x20;

若因 Atlas 自身运营原因（包括但不限于航司侧账户余额不足、支付通道限制等），导致系统在出票过程中主动加收费用，且客户在知情并确认加价后完成购票并成功出票的，Atlas 将在出票完成后，按实际加收金额退还该笔额外费用。&#x20;

该退还属于 Atlas 对自身运营原因导致额外收费的纠正，不构成任何形式的服务赔付，且与客票后续是否使用、退票、改期无关。 &#x20;
