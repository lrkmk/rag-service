# 扩展与集成

使用此部分获取 Atlas 在主预订和预订后流程之外的扩展与集成指导。

当您需要以下内容时从这里开始：

* 添加 webhook 或通知投递
* 在交易流程之外使用 Atlas 数据
* 实现航司特定或特殊场景的集成

### 主要部分

#### 通知与 Webhook 投递

使用 [Webhook 概述](/api-wen-dang/product-guides/extensions-and-integrations/webhook-overview.md) 了解事件覆盖范围和处理方式。

使用 [多渠道通知](/api-wen-dang/product-guides/extensions-and-integrations/multi-channel-notifications.md) 进行渠道设置和投递选项配置。

#### 航班数据与搜索支持

使用 [航班数据源](/api-wen-dang/product-guides/extensions-and-integrations/flight-data-feed.md) 在需要定时运价投递以构建自有数据层时。

#### 特殊项目

使用 [特殊集成](/api-wen-dang/product-guides/extensions-and-integrations/special-integrations.md) 在业务场景需要航司特定或居民票价逻辑时。

### 需要精确的端点字段？

使用 [API 参考](/api-wen-dang/api-reference.md)。
