# 座位回退模式

使用此页面了解座位履约回退模式。

### 概述

当座位选择包含在订单中时，您可以定义 Atlas 在座位履约失败时应如何处理。

### 支持的模式

* `STOP_TICKET` — 停止出票，取消整个订单并退款
* `STOP_SEAT` — 出票，移除座位并退还座位金额
* `SIMILAR_SEAT` — 出票，由运营选择类似座位

### 选择模式

* 如果座位是强制性的，使用 `STOP_TICKET`
* 如果座位是可选的但优先选择，使用 `STOP_SEAT`
* 如果任何座位都可以接受，使用 `SIMILAR_SEAT`
