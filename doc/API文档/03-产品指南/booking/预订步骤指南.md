# 预订步骤

使用此部分获取每次解释一个预订操作的步骤指南。

每个页面涵盖一个步骤、API 入口点或操作交接。

这些页面可在多个预订流程中重复使用。

它们本身并非固定的端到端顺序。

当您需要完整的预订顺序时，请先使用流程页面。

在您已经知道要构建哪个预订流程后，使用此部分。

当您需要以下内容时从这里开始：

* 详细了解一个预订步骤
* 将任务映射到正确的端点序列
* 构建或排查搜索、验价、订单、支付或订单跟进

`核价出票` 是[核价出票接口](/api-wen-dang/product-guides/booking/booking-flows/fulfillment-flow.md)流程的入口步骤。
