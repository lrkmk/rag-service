# 产品指南

使用此部分可在实现端点详情之前选择正确的指南。

{% hint style="warning" %}
**🆕 新产品：** 当您需要 `getOfferPrice.do` 预订路径、更广泛的展示规则以及创建订单后立即支付时，请从[🆕 核价出票接口](/api-wen-dang/product-guides/booking/booking-flows/fulfillment-flow.md)开始。

在两种基于报价的路径之间做选择时，请使用[Get Offer vs 核价出票接口](/api-wen-dang/product-guides/booking/booking-overview/booking-decisions/get-offer-vs-get-offer-price.md)。
{% endhint %}

浏览产品指南的三个部分：

### 预订

此部分涵盖完整的预订领域。

从[预订](/api-wen-dang/product-guides/booking.md)开始。

如果您想要最新的预订路径，请从[🆕 核价出票接口](/api-wen-dang/product-guides/booking/booking-flows/fulfillment-flow.md)开始。

然后使用：

* [预订概述](/api-wen-dang/product-guides/booking/booking-overview.md)了解共享规则和标识符
* [预订路径](/api-wen-dang/product-guides/booking/booking-flows.md)选择端到端预订路径
* [预订步骤](/api-wen-dang/product-guides/booking/booking-step-guides.md)了解各步骤预订页面
* [可选附加服务](/api-wen-dang/product-guides/booking/optional-ancillaries.md)了解可选的座位和行李

### 预订后

此部分涵盖退款、作废、订单维护和 PNR 跟进。

从[预订后操作](/api-wen-dang/product-guides/post-booking.md)开始。

### 扩展与集成

此部分涵盖通知、数据源和特殊集成。

从[扩展与集成](/api-wen-dang/product-guides/extensions-and-integrations.md)、[Webhook 概述](/api-wen-dang/product-guides/extensions-and-integrations/webhook-overview.md)或[航班数据源](/api-wen-dang/product-guides/extensions-and-integrations/flight-data-feed.md)开始。

### 需要精确字段时请使用 API 参考

需要精确的架构、参数和端点级示例时，请使用[API 参考](/api-wen-dang/api-reference.md)。

### 需要集成步骤时请使用集成指南

需要沙箱设置、开发、UAT 和生产上线时，请使用[集成指南](/api-wen-dang/readme-1.md)。
