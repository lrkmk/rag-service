# 航班数据源

使用此页面集成 Atlas 航班数据源。

### 概述

航班数据源提供对 Atlas 航班数据的编程访问，使您能够检索航线、时刻表和航班可用性信息。

### 主要功能

* 检索航班时刻表
* 查询航线覆盖
* 获取航班状态更新
* 访问历史航班数据
