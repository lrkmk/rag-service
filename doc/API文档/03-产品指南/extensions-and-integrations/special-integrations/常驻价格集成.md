# 常驻价格集成

使用此页面集成居民价格功能。

### 概述

居民价格集成允许特定航线的居民享受折扣票价。

### 关键规则

* 居民价格适用于特定航线的居民
* 需要验证居民身份
* 价格可能显著低于标准票价
