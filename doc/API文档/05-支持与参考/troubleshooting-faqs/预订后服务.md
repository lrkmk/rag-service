使用本页面了解出票后操作，包括退款、作废和附加服务问题。

当你需要以下内容时，从这里开始：

* 了解退款流程如何运作
* 决定是退款还是作废订单
* 处理出票后附加服务或订单对账

### 常见问题

#### 退款流程如何运作？

退款流程涉及请求退款报价、检查有效性和提交流程。

退款报价不会锁定价格。一旦提交，Atlas 将向航空公司发起退款。

退款的处理时间因航空公司而异。

### 退款流程如何运作？

退款流程：

1. 请求退款报价
2. 检查有效性
3. 提交退款请求

退款报价不会锁定价格。一旦提交，Atlas 将向航空公司发起退款。退款的处理时间因航空公司而异。

#### 何时应使用作废而不是退款？

当您需要在航空公司作废窗口内取消未使用的机票时，使用作废。

作废通常更快且成本更低。

一旦作废窗口关闭，使用退款流程。

### 何时应使用作废而不是退款？

当您需要在航空公司作废窗口内取消未使用的机票时，使用作废。\
作废通常更快且成本更低。\
一旦作废窗口关闭，使用退款流程。

#### 退款报价在提交前会过期吗？

是的。退款报价在创建后有效期为 3 天。

如果过期，请求新的退款报价。

#### 如何在出票后添加附加服务？

当原始预订使用 Atlas API 出票后，使用出票后附加服务流程。

从订单查询验证出票状态，然后在支持时继续添加行李或座位。

### 如何在出票后添加附加服务？

从订单查询验证出票状态，然后在支持时继续添加行李或座位。

#### 出票后附加服务支持退款吗？

是的。\
出票后附加服务遵循与主机票相同的退款流程。

您需要请求退款报价并提交退款请求。

### 如何对账预订后交易？

使用 ATRIP 对账工具。\
访问 `Finance` -> `Settlement`。

#### 退款金额记入哪里？

对于 **VCC 直通**支付，退款资金通常返回到原始卡。\
对于 **存款**支付，Atlas 在收到航空公司资金后记入您的余额。

#### 可以部分退款吗？

部分退款取决于航空公司政策。

并非所有航空公司都支持部分退款。

退款报价响应将指示哪些乘客或航段符合退款条件。

#### 作废和退款之间有什么区别？

作废在航空公司作废窗口内取消未使用的机票。它通常更快且相关费用更少。

退款适用于已出票的机票，特别是当作废窗口已关闭时。

#### 如何检查作废窗口是否仍然可用？

在航空公司出票后尽快尝试作废。

如果 Atlas 返回 `822`（超过机票作废截止时间），则作废窗口已关闭，应使用退款流程。
