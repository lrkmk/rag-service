使用本部分获取跨集成流程重复使用的稳定参考数据。

当你需要以下内容时，从这里开始：

* 在实施前确认共享请求值
* 查看可重复使用的枚举值或查找值
* 查找多个工作流使用的辅助参考页面

### 常见问题

#### 参考数据用于什么？

使用本部分获取跨 Atlas API 流程重复使用的稳定共享值和查找数据。

#### 何时应打开参考数据而不是工作流页面？

当你已经知道工作流，只需要辅助值、枚举值或查找数据时，打开参考数据。
