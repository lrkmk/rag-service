# 如何申请非自愿退票？

您可以通过ATRIP申请非自愿退票。

**退票路径：**

**ATRIP -> 订单中心 -> 我的订单 -> 订单详情 -> 取消&退票**

<figure><img src="/files/joqHelrJzX58qil64A0V" alt=""><figcaption></figcaption></figure>

**操作说明：**

1.点击"取消&退票" 按钮

2.选择需要退票的具体乘客<br>

<figure><img src="/files/1NFQTzEgSkyyB4LobzK8" alt=""><figcaption></figcaption></figure>

3.选择退票/退款方式，Atlas支持以下两种方式，请根据需求选择：

* 如果您需要Atlas代您向航空公司提交退票请求，请点击按钮【**尚未操作，请Atlas操作退票**】；
* 已自行退票：如果您或乘客已经自己向航空公司提交了退票请求，并且得到了航空公司的确认，请点按钮【**航空公司已退票，请Atlas退款**】**，**&#x41;tlas将在收到航空公司的退款后立即为您办理退款。

{% hint style="warning" %}
对于使用VCC Passthrough支付方式的订单(可以通过订单详情里的“支付渠道：虚拟信用卡”来辨识），不支持选择“**航空公司已退票，请Atlas退款**”选项。因为在客户使用自己的信用卡支付时，信息传递只发生在客户和航空公司之间，Atlas无法了解实际退款金额的到账情况。
{% endhint %}

<figure><img src="/files/ejDE8trfJT5Jep3xLnvW" alt=""><figcaption></figcaption></figure>

4.提交退票/退款申请

\ <mark style="background-color:orange;">**1）选择【尚未操作，请Atlas操作退票】**</mark>

Atlas提供退现金及退券两种方式（具体以航司政策为准），请选&#x62E9;**【非自愿退票】**&#x540E;，按页面提示完成操作，并确认提交申请即可。<mark style="color:red;">**注意：退票申请提交后将无法撤回，请谨慎操作。**</mark>

* **退现金页面：**

![](/files/jeZeu7qk9FRoLitdci1t)

{% hint style="warning" %}
若Atlas已检测到航变，则无需上传凭证，若Atlas未检测到航变，则需要上传凭证。
{% endhint %}

* **退券页面：**

![](/files/kBi75MPpiJYEQRYy69Hr)

![](/files/QOCPwsgsgpbTmH7YF3W6)

<mark style="background-color:orange;">**2）选择【航空公司已退票，请Atlas退款】**</mark>

请根据页面说明，选&#x62E9;**【非自愿退票】**，填写航司履约日期（即航司确认退票的日期），上传航司履约证明，然后点击“确认”按钮提交申请。

<figure><img src="/files/f7nba6ePYCpfshe2Ixtm" alt=""><figcaption></figcaption></figure>

5.提交成功后，Atlas将收到并处理您的退票/退款申请，同时您的申请记录会展示在“退票信息”和“退票单”页面，您随后可以在任意一个页面跟踪处理进度。
