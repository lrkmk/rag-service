# 如何用券出新票？

Atlas目前仅受理航空公司 **VJ** 的用券场景。

关于 **VJ** 用券问题，您可以提交工单 ，并写明用券需求，Atlas将安排工作人员协助您妥善处理。

**提交路径**：ATRIP -> 订单中心 -> 我的订单 -> 订单详情 -> Eva -> 票务咨询 -> 出票-> 用券出新票
