# Atlas 废票服务快速指南

Atlas 废票（Void）服务相关文档分别说明了服务适用范围、收费及责任边界，以及 ATRIP 和 API 两种使用方式下的操作流程。您可根据实际需要查看对应文档。

### 一、提交申请前：确认适用范围及服务规则

#### 1. Atlas 废票政策说明

用于了解废票服务的适用范围及基础处理规则，包括：

* 支持废票的场景；
* 暂不支持废票的场景；
* 废票申请窗口期；
* 废票结果及退款相关说明。

点击查看：[《Atlas 废票政策说明》](https://resources.atriptech.com/bang-zhu-zhong-xin/re-men-fen-lei/shou-hou-piao-wu/fei-piao/atlas-fei-piao-zheng-ce-shuo-ming)

#### 2. Atlas 废票服务条款

用于了解使用废票服务时的收费规则、服务责任及相关边界，包括：

* Atlas 的服务范围及责任边界；
* 废票结果判定及相关责任边界；
* 航司拒绝、废票失败及退款说明；
* 服务费标准及收费触发条件。

点击查看：[《Atlas 废票服务条款》](https://resources.atriptech.com/bang-zhu-zhong-xin/fu-wu-tiao-kuan/atlas-void-service)

### 二、提交申请及查询处理进度

确认订单符合废票条件并了解相关规则后，您可根据实际使用方式查看对应操作指引。

#### 1. 通过 ATRIP 操作

如您通过 Atlas 平台页面提交废票申请，请点击查看 [《如何申请废票？》](https://resources.atriptech.com/bang-zhu-zhong-xin/re-men-fen-lei/shou-hou-piao-wu/fei-piao/ru-he-shen-qing-fei-piao)，了解：

* 如何提交废票申请；
* 如何查看处理状态及进度；
* 如何查看废票结果及相关订单信息。

#### 2. 通过 API 操作

如您需要将废票服务接入自有系统，请点击查看 [《废票 API 文档》](https://resources.atriptech.com/api-wen-dang/api-can-kao/yu-ding-hou-api/fei-piao)，了解：

* 废票报价及可废票判断；
* 废票申请提交；
* 废票状态查询；
* Webhook 通知；
* API 接入要求及参数说明。
