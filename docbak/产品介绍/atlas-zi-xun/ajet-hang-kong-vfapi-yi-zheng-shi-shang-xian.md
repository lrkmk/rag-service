# AJet 航空（VF）API 已正式上线

舆途科技（Atlas）成为全球首家基于 AJet（VF）最新 NDC 24.1 标准正式上线其数据的合作伙伴。

现通过您现有的 API 连接，即可直接接入土耳其增长最快的航司之一 AJet 的全部票价、辅营服务及具有竞争力的国际票价。

无需额外对接，也无需新增集成。只是多了一家可以立即开卖的新航司。

## 为什么值得关注？

AJet 运营覆盖 40 个国家、100 多个目的地，以伊斯坦布尔萨比哈·格克琴机场（SAW）和安卡拉埃森博阿机场（ESB）为枢纽。

作为土耳其国内市场占有率达 37% 的航司，AJet 正积极扩展欧洲、巴尔干半岛、中东及中亚市场 —— 这些也是旅客持续增长、搜索热度较高的区域。

同时，作为首家上线 AJet 最新 NDC v24.1 的合作伙伴，舆途科技（Atlas）可帮助航旅商家们更早获得可零售化的航司数据，在更广泛市场开放前抢先布局。

## 您现在可获得的能力

### 一、AJet 全航线网络

在舆途科技（Atlas）上预订 AJet 完整的国内及国际航线网络——覆盖 100% 直达航线及关键中转市场，预订窗口为 120 天。所有航线均可通过您现有的 API 连接，直接获取。

{% hint style="success" %}
**独家优势**： AJet 纯国内航线仅通过其官网和舆途科技（Atlas）两个渠道开放。目前没有任何其他 API 合作伙伴能够获取这部分数据。通过我们的 API，您将拥有其他分销平台无法提供的独家国内航线库存。
{% endhint %}

### 二、预订流程内支持辅营销售

AJet 辅营产品现已开放，可直接嵌入您的零售与服务流程，帮助提升客户体验，并释放更多辅营收益机会。

当前支持：

* 票价系列（Fare Family）
* 选座服务
* 托运行李

### 三、有价格信心的预订

国际票价与 AJet 直销渠道保持一致。

国内航线每段包含 USD 2–3 的代理加价，这是 AJet 对所有 API 分销合作伙伴的统一政策，并非仅针对舆途科技（Atlas）加收。

但需要强调的是，尽管加价政策与其他供应商相同，我们是唯一能够获取 AJet 全部纯国内航线数据的合作伙伴。

简而言之，您以相同的成本，获得了其他渠道根本不存在的独家国内航线库存。

### 四、所有现有客户已自动开通

AJet 采用预审批分销模式。所有舆途科技（Atlas）的客户均已获得访问授权——您无需任何操作即可在连接中激活 AJet。

## AJet 数据现已支持搜索与预订

AJet 已正式开放搜索及预订能力。如有任何疑问、希望探讨如何最大化利用这些数据，或者目前缺少您认为重要的中转航线希望我们优先添加，请联系您的客户经理。

一个 API，多一家正在快速扩张的航司。

希望这能帮助您在土耳其、欧洲及中亚市场获得更多增长机会。
