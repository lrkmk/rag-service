#### 三、从报价到履约的完整能力&#x20;

不仅支持选座报价，更支持完整选座履约能力。&#x20;

#### 四、灵活的选座失败处理机制&#x20;

当旅客选择的座位已被占用时，可根据业务场景选择三种策略：&#x20;

1. 指定座位优先 → 座位失败则取消出票&#x20;
2. 出票优先 → 正常出票，退还选座费用&#x20;
3. 履约优先 → 自动匹配相似座位&#x20;

合作伙伴可通过 API 在订单创建时指定对应策略。&#x20;

### 常问问题

<details>

<summary>哪些场景暂不支持选座服务？</summary>

目前不支持纯选座询价、非舆途科技（Atlas）订单和出票后二次选座服务。如有特殊业务需求，欢迎联系您的客户经理进一步沟通。

</details>

<details>

<summary>如果旅客退票，选座费可以退吗？</summary>

**自愿退票：**&#x65C5;客主动申请退票时，选座费通常不予退还。大部分航司均将选座视为已消费辅营产品，不支持退款。&#x20;

**非自愿退票：**&#x5982;因航班取消、航班时刻变更等航司原因导致退票，选座费用将按照航司政策与机票一并处理。&#x20;

</details>

<details>

<summary>选座成功后，是否保证最终一定能履约？</summary>

选座属于高实时性业务，座位库存、价格及最终座位分配均由航司控制。&#x20;

舆途科技（Atlas）提供选座查询、报价及履约能力，并持续优化履约成功率，但无法保证最终座位一定成功履约。如遇机型更换、航司改配座位等情况，将以系统返回结果与航司政策为准。&#x20;

</details>
