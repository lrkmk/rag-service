# 维兹航空（Wizz Air） API 直连正式上线舆途科技（Atlas）

如果您已经在通过舆途科技（Atlas）销售维兹的机票，那么您的数据迎来了一次重大升级。

维兹（Wizz Air，W6 / W4 / W9）现已通过 API 直连正式上线——通过您现有的舆途科技（Atlas）连接，即可获得更稳定、更适配分销场景的接入能力，直连欧洲最大的超低成本航司之一。

对您当前的销售没有任何影响，只是底层的支撑更强了。

### 为什么这件事很重要

维兹的航线网络覆盖 45 个国家、185 个目的地，在中东欧、中东、中亚及北非地区覆盖尤为密集。

这些都是客户正在搜索的高需求航线——现在，您可以直接通过舆途科技（Atlas） API 完成预订。

这是一次直连分销合作，意味着数据直接从维兹获取——更稳定、出票失败率降低、履约能力更强。

您现在可以做什么

#### 一、接入维兹完整航线网络

支持预订维兹旗下三个运营实体（W6、W4、W9）的去程与往返航班，覆盖欧洲、中东、中亚及北非的直飞航线。成人、儿童、婴儿票价全部支持。

#### 二、在预订流程中直接销售辅营产品

维兹的辅营产品可在订票时一并完成销售——无需额外步骤，也无需出票后补操作。包括：

* Fare Families
* 选座
* 托运行李
* 含优先登机的大型手提行李套餐

#### 三、最长可预订未来 90 天机票

上线即开放 90 天预订窗口，后续计划扩展至 180 天——为您预留充足的行销和促销空间。

#### 四、票价与维兹官方渠道一致

无加价。您看到的价格，就是维兹实际销售的价格。

### 温馨提示

出票后如需增购辅营产品，可通过 wizzair.com 使用 PNR 和乘客姓氏直接操作——无需登录维兹账户。

旅客信息更正、改期及退票，需通过维兹账户登录处理——由乘客自助完成。

确保客户可以直接接收维兹发送的所有原始通知（航班变更、异常提醒、服务公告等），且不得修改内容。

为正式启用您的接入权限，您需要完成维兹要求的条款与条件（T\&C）签署。
