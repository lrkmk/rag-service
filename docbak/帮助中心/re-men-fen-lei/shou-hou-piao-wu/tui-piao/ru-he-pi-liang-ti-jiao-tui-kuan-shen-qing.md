# 如何批量提交退款申请？

如果您或乘客已自行向航空公司提交了退票申请并获得确认，涉及多笔订单时，可通过ATRIP的批量提交退款申请功能。Atlas将在收到航司退款后自动为您处理退款。

{% hint style="warning" %}
据国际航协（IATA）惯例，机票有效期通常为出票后1年。超过有效期后，Atlas 将无法再受理相关退款申请。 对于同时满足以下条件的未完结 Refund Claim（退款申请），系统将自动拒绝并完结： &#x20;

* 自出票日期起已满1年；&#x20;
* 自客户提交退款申请起已满1个月。
  {% endhint %}

**提交路径：**&#x41;TRIP -> 订单中心 -> 退票单 -> 批量提交退款申请

<figure><img src="/files/CxCoAteK4xMzVO6eNhu0" alt=""><figcaption></figcaption></figure>

**操作说明：**

1. 点击“批量提交退款申请”按钮，进入批量提交页面，下载批量提交的Excel模板

![](/files/OufnZs8BPMJzeiJSm1uz)

2. 将需要批量提交退款的订单号、退票类型和自行完成履约的日期填入Excel模板中

*<mark style="color:blue;">注：请按表格示例格式填写相关内容，填写后请删除示例部分</mark>*

![](/files/6ccWcqfGhki64V9o8kzv)

3. 通过“选择文件”入口上传已经填写好的Excel本地文件

*<mark style="color:blue;">注：每次仅可导入一个文件，且单文件数据量不超过1000条</mark>*

![](/files/8p8gpBycHH96hd5vHfw8)

4. 文件上传后，可在页面下方的「导入记录」中查看上传状态。若导入失败，请下载失败报告并参照错误提示调整后重新导入。

![](/files/CEVgKkS1V9juubkPDEK2)

5. 批量提交退款操作成功后，可&#x5728;**「退票单」**&#x6A21;块查看退款处理进度。 Atlas将在收到航空公司退款后，自动退还给您。

![](/files/8L4KS6xUBuFnCClbWbQo)

6. 如果您对退款进度有疑问，可以通过每个退票单的「确认退款进度」入口，上传航司确认退款的凭证，然后联系Atlas协助查询该笔交易的退款情况。

*<mark style="color:blue;">注：凭证上传成功后，退票单号左上角将会展示「有凭证」的标识，点击操作中的</mark><mark style="color:blue;">**「查看详情」**</mark><mark style="color:blue;">按钮便可查看凭证内容。</mark>*

![](/files/TXurFfvdWcPwBels39i1)
