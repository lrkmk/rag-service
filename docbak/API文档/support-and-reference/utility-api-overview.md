本部分汇总运营和支持团队使用的非交易类 API。

当您需要以下能力时，从这里开始：

* 检查余额或财务状态
* 查询运营相关邮件记录
* 导出航线数据
* 生成 ATRIP 访问令牌

### FAQ

#### 这些工具主要用于什么？

它们支持财务检查、邮件查询、航线导出和 ATRIP 访问等非交易类流程。

它们不属于从搜索到出票的主预订链路。

#### 我应该先打开哪个工具页面？

财务检查使用**余额**，邮件记录查询使用**邮件查询**，航线数据提取使用**航线导出**，令牌生成使用**ATRIP 令牌**。
